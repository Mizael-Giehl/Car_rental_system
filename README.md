
## Car rental system - JDBC

Project developed to practice: - JDBC (Java Database Connectivity).

## Features
- Register users.
- Register, list, and delete cars.
- Register new models.
- Export cars to a spreadsheet.
- Import cars from a spreadsheet.
- Add balance.
- Rent a car.
- Return the car.

## Technologies 
- Java 25
- MySQL
- JDBC

## Applied concepts 
- JDBC
- Connection
- ResultSet
- PreparedStatement
- IOException
- SQLException

## Execution 
Open the project in NetBeans and run it:
Main.java
